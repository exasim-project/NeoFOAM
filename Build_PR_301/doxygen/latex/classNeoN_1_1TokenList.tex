\doxysubsection{Detailed Description}
A class representing a list of tokens. 

The \doxylink{classNeoN_1_1TokenList}{Token\+List} class provides functionality to store and manipulate a list of tokens. It supports insertion, removal, and retrieval of tokens of any type using std\+::any.
