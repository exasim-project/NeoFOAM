Creates an instance of a derived class based on the provided key.

This function creates an instance of a derived class based on the provided key. The key is used to look up a factory function in a table, and the factory function is then called with the provided arguments to create the instance.

Parameters
keyThe key used to look up the factory function in the table.
argsThe arguments to be forwarded to the factory function.
Returns
A unique pointer to the created instance.
Exceptions
std::out_of_rangeif the provided key does not exist in the table.
