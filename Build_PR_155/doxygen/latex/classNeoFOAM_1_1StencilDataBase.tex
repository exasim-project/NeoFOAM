\doxysubsection{Detailed Description}
A class that represents a stencil database. 

The \mbox{\hyperlink{classNeoFOAM_1_1StencilDataBase}{Stencil\+Data\+Base}} class provides a container for storing stencil data. It allows insertion, retrieval, and checking of stencil data using string keys.
